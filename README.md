# Automated WhatsApp Matchmaker

A complete system for an automated matchmaking business, featuring:
* Next.js internal dashboard
* Firebase Auth & Firestore backend
* WhatsApp Cloud API Webhook
* Gemini AI integration for processing requirements

## Setup
1. Clone this repository.
2. Run `npm install`.
3. Rename `.env.local.example` to `.env.local` and add your keys.
4. Run `npm run dev` to start the server.
